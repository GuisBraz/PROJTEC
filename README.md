# PROJTEC 1D

App de otimização de corte linear (1D) e aproveitamento de barras/laminados.

## Como rodar no seu computador

1. Instale o Node.js (https://nodejs.org) — versão LTS.
2. Abra o terminal dentro desta pasta.
3. Rode:

   npm install
   npm run dev

4. Abra o endereço que aparecer no terminal (geralmente http://localhost:5173).

## Como publicar como site de verdade

Veja o passo a passo no chat com o Claude, ou:
1. Suba esta pasta para um repositório no GitHub.
2. Crie uma conta em vercel.com, conecte com o GitHub e escolha o repositório.
3. A Vercel detecta automaticamente que é um projeto Vite/React e publica sozinha.
4. Você recebe um link (ex: projtec-1d.vercel.app), que pode trocar por um domínio próprio depois.
